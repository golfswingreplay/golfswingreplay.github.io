Beta tester onboarding

Launch AppStore

You can only download beta Golf Swing Replay application through TestFlight.

Launch the AppStore & search for TestFlight application.

Install TestFlight

Install TestFlight as you would with any other app.

Check your email

Once you are approved to become a beta tester, you will receive an invitation from TestFlight that looks like the one on the left.

Check your spam!

If you can't find the email, make sure you check your spam/junk folder.

Find out your unique redeem code

Click on View in TestFlight to be taken to the website to get your unique code.

Grab your code

You will need to either note down or copy the unique to you code.

Start TestFlight

Find TestFlight on your phone and start it up.

Redeem your code

In the top right corner, you will find a redeem button. Click it to be prompted for your code.

Type in your code

Type in your code you saved earlier and press OK to get Golf Swing Replay app.

Install the app

Golf Swing Replay application should now appear on the list of applications you have access to beta test.

Enjoy the app!

You now have access to the latest of the Golf Swing Replay. Once again, big thanks and enjoy your journey of becoming a better golfer.
