Beta test install

Launch AppStore

You can only download the Golf Swing Replay beta version via Apple's TestFlight.

Launch the AppStore & search for TestFlight application.

Install TestFlight

Install TestFlight as you would with any other app.

Check your email on your iOS device

Once you are approved to become a beta tester, you will receive an invitation from TestFlight that looks like the one on the left.

Check your spam!

If you can't find the email, make sure you check your spam/junk folder.

Install the app

Golf Swing Replay application should now appear on the list of applications you have access to beta test.

Enjoy the app!

You now have access to the latest of the Golf Swing Replay. Once again, big thanks and enjoy your journey of becoming a better golfer.

Install TestFlight on secondary iOS device

If both devices use the sample Apple ID then Golf Swing Replay should now appear on the list of applications you have access to beta test.

Using two devices with different Apple IDs

If the second devices uses a different Apple ID, you'll need to request access for that email address. Just let us know the secondary Apple ID email address, and we will send an additional invite
